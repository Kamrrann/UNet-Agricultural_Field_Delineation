
# Sentinel-2 Agricultural Field Delineation using U-Net

A U-Net deep learning model for precise agricultural field boundary delineation using TensorFlow and Keras. This prepository provides end-to-end workflows from training to prediction, including visualization of segmentation masks.

## Overview
Accurate delineation of agricultural fields is essential for crop monitoring, yield estimation, and effective land management. This repository implements a  U-Net architecture for multi-class segmentation of agricultural fields using Sentinel-2 imagery. The model is designed to distinguish between different land cover types within agricultural landscapes, improving the precision and reliability of field boundary detection.
## Dataset
 The model is trained and evaluated on the SDG2 Field Delineation Dataset provided by SustainLab. This dataset contains Sentinel-2 agricultural field imagery suitable for segmentation tasks.

•	Source: SustainLab SDG2 Field Delineation Dataset

•	Data type: Sentinel-2 imagery 

•	Task: Multi-class segmentation for agricultural field boundary delineation 

The dataset includes annotated field boundaries that enable the model to learn precise delineation and classification of different land cover types within agricultural landscapes.
